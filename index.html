CanvasJS Chart - Technical Indicators

This plugin allows you to add technical indicators to CanvasJS chart. Check out Github Repository for more info / examples.

How to Use?

  • Enable all the Technical Indicators under Chart-Options
  • Render the chart
var chart = new CanvasJS.Chart("chartContainer", {
    .
    .
    .
    technicalIndicators: {
        sma: {
            enabled: true,
            linkedDataSeriesIndex: 0,
            period: 7,
            color: "green",
            showInLegend: true,
            name: "SMA"
        },
        ema: {
            enabled: true,
            linkedDataSeriesIndex: 0,
            period: 7,
            color: "red",
            showInLegend: true,
            name: "EMA"
        }
    },
    //Chart Options
    .
    .
    .
});
chart.render();
